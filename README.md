# gemini-auth-nano-banana
A  skill for gemini and nano banana editing with Google account authentication. Generate new images from text plus one or more source images without using an API key. Good for redraws, background replacement, pose changes, and style-guided edits. / 一个调用gemini与nano banana的 skill，通过 Google 账号登录态而非 API key，结合文字与一张或多张参考图生成新图片，适合重绘、换背景、姿势变化和风格化编辑。
